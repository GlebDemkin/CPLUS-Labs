Название
Выравнивание текста по центру с переносом слов (C++ / STL)

Описание
Программа читает текст из файла и выводит его, формируя строки длиной не более L и выравнивая каждую строку по центру.
Если исходные строки файла длиннее L, лишние слова переносятся на следующую строку.
Если встречается фрагмент без пробелов, который длиннее L (например, одно очень длинное слово), он разрезается на части длиной L.

Ввод

1) Текстовый файл (с любым количеством строк).
2) Параметры командной строки:
	argv[1] — имя входного файла
	argv[2] — длина строки L (целое положительное число)

Запуск
center_align <filename> <L>

Пример:
center_align input.txt 40

Если аргументы не указаны, рекомендуется выводить подсказку:
Usage: center_align <filename> <line_length>

Вывод
Отформатированный текст печатается в консоль.

Правила формирования:
- каждая выводимая строка имеет длину не больше L
- строки выравниваются по центру путём добавления пробелов слева
- слова не разрываются, если их длина <= L
- если слово/фрагмент без пробелов имеет длину > L, он режется на части по L символов

Требования к реализации:
- Чтение файла построчно: std::getline и std::ifstream
- Разбиение строки на слова: std::istream_iteratorstd::string
- Использование STL: std::string, std::vector и т.п.
- Использовать range for там, где это возможно: for (auto &w : words) { ... }
-  Для продвинутых: вместо “голого” std::vectorstd::string
 можно использовать классы-обёртки с конструкторами и методами

Алгоритм работы (кратко, но по шагам)

1) Прочитать argv[1] и argv[2], проверить что L > 0.
2) Открыть файл через ifstream, проверить успешность открытия.
3) Читать файл построчно через getline.
4) Каждую считанную строку разбить на слова через istream_iterator<string>.
5) Последовательно добавлять слова в буферную строку:
	- если слово помещается (с учётом пробела), дописать его
	- иначе сохранить буферную строку в результат и начать новую
6) Если слово длиннее L:
	- сохранить текущий буфер (если есть)
	- разрезать слово на куски длиной L и вывести/сохранить их как отдельные строки
7) Для каждой готовой строки выполнить центрирование:
	- pad = (L - length) / 2, если length < L
	- вывести pad пробелов + строку
